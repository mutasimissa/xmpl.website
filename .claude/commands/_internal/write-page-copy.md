Read `skills/page-copy/SKILL.md` and follow every step.

1. Read the page brief from `content/02-page-briefs/`.
2. Read brand strategy (tone), value proposition, SEO brief (keywords), and relevant blueprint from `agency/blueprints/`.
3. Write copy section by section: H1 with primary keyword, body with proof, CTAs matching business strategy.
4. Meta description must include primary keyword + action verb.
5. Append to `content/04-content-deck.md` under `## Page Name`.
6. Score against `agency/rubrics/copy-rubric.md` — minimum average 4.
