# Launch Checklist

## 1. Business alignment

- [ ] Positioning is consistent across all pages (headline, subheadline, about section)
- [ ] Offers on the site match the business model file exactly
- [ ] Each target persona is addressed in the copy
- [ ] No contradictions between business files and site content
- [ ] Brand tone of voice is consistent across all pages

## 2. UX and navigation

- [ ] Visitor understands the business within 5 seconds of landing on the homepage
- [ ] Navigation matches the sitemap and is intuitive on desktop and mobile
- [ ] Page hierarchy follows the buyer journey (awareness to action)
- [ ] Mobile experience is fully functional with touch-friendly targets (44x44px minimum)
- [ ] Skip-to-content link is present and visible on focus

## 3. SEO readiness

- [ ] Every page has a unique title tag (50-60 chars) and meta description (150-160 chars)
- [ ] Primary keyword appears in H1 and first paragraph of each page
- [ ] Schema markup is implemented per the SEO brief (Organization, Service, FAQPage, BreadcrumbList)
- [ ] Internal links create logical pathways between pages per the linking plan
- [ ] XML sitemap route and robots.txt are configured and accessible

## 4. Technical foundation

- [ ] All pages render correctly on desktop, tablet, and mobile
- [ ] No broken links, missing images, or console errors
- [ ] Forms submit correctly and show confirmation or error states
- [ ] Core Web Vitals targets are met (LCP < 2.5s, INP < 200ms, CLS < 0.1)
- [ ] HTTPS is enforced and canonical URLs resolve correctly

## 5. Conversion readiness

- [ ] Primary CTA is visible above the fold and consistent across pages
- [ ] Contact page is low-friction with clear next steps
- [ ] Trust signals (credentials, process, proof) appear near CTAs
- [ ] Objections are handled on relevant pages (FAQ, process explanation, social proof)

## 6. Accessibility

- [ ] Semantic HTML structure with proper heading hierarchy (one H1, H2 for sections)
- [ ] Keyboard navigation works on all interactive elements (Tab, Shift+Tab, Enter, Escape)
- [ ] Color contrast meets WCAG 2.2 AA (4.5:1 for normal text, 3:1 for large text)
- [ ] All images have descriptive alt text
- [ ] Every form input has an associated label and clear error messaging

## 7. Trust and compliance

- [ ] Privacy policy is published and linked from the footer
- [ ] Cookie consent is implemented if required by target market regulations
- [ ] Contact information (email, phone, address) is visible on the site
- [ ] Legal disclaimers are present where needed (terms, service conditions)
- [ ] No unverified claims, logos, or testimonials on the site
