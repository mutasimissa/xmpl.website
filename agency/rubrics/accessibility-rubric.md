# Accessibility Rubric

Score each item from 1 to 5.

| Criterion | 1 (poor) | 3 (acceptable) | 5 (excellent) |
|---|---|---|---|
| **Semantic structure** | No semantic HTML, divs only | Some semantic elements, inconsistent | Full semantic HTML with landmarks and proper hierarchy |
| **Keyboard navigation** | Not keyboard accessible | Most elements reachable, some traps | Fully keyboard navigable with visible focus indicators |
| **Color contrast** | Fails WCAG AA on multiple elements | Passes on body text, fails on some UI | Passes WCAG AA on all text and interactive elements |
| **Image accessibility** | No alt text on any images | Some alt text, inconsistent quality | All images have appropriate, descriptive alt text |
| **Form accessibility** | No labels, no error handling | Labels present, weak error messages | Full labels, clear errors, ARIA announcements |
| **Screen reader support** | Unusable with screen reader | Mostly navigable, some gaps | Fully usable, logical reading order, live regions |

A page is not ready for launch below an average of 3.
