# Brand Identity Rubric

Score each item from 1 to 5.

| Criterion                | 1 (poor)                                    | 3 (acceptable)                                | 5 (excellent)                                                |
| ------------------------ | ------------------------------------------- | --------------------------------------------- | ------------------------------------------------------------ |
| **Palette coherence**    | Random colors with no relationship          | Colors related but inconsistent in tone       | Harmonious palette derived from brand direction              |
| **Contrast compliance**  | Fails WCAG AA on primary text combos        | Partial compliance, some combos fail          | All text/background combinations pass WCAG AA                |
| **Token completeness**   | Only colors defined                         | Colors and typography defined                 | Full token set: colors, typography, spacing, radii, shadows  |
| **Philosophy alignment** | Tokens contradict the stated philosophy     | Loosely related to philosophy                 | Tokens clearly express and reinforce the brand philosophy    |
| **Logo integration**     | No logo rules or file paths                 | Logo listed but no usage guidance             | Logo variants, clear space, and usage rules fully defined    |
| **RTL readiness**        | No RTL font consideration                   | RTL fonts mentioned but not specified         | RTL fonts specified per locale with appropriate fallbacks    |

A brand identity output is not ready below an average of 4.
