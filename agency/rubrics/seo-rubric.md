# SEO Rubric

Score each item from 1 to 5.

| Criterion                | 1 (poor)                                | 3 (acceptable)                 | 5 (excellent)                                                              |
| ------------------------ | --------------------------------------- | ------------------------------ | -------------------------------------------------------------------------- |
| **Intent alignment**     | Keywords don't match buyer intent       | Some alignment but gaps        | Every keyword maps to a clear buyer intent stage                           |
| **Keyword targeting**    | No keywords assigned to pages           | Keywords assigned but generic  | Specific, commercial keywords with realistic ranking potential             |
| **Metadata readiness**   | No metadata planned                     | Some title tags drafted        | Complete title + description for every page, unique and keyword-rich       |
| **Internal link logic**  | No internal linking plan                | Some links but no strategy     | Logical link graph supporting buyer journey and SEO value flow             |
| **On-page structure**    | No heading hierarchy or structure       | Some structure, inconsistent   | Clean H1-H3 hierarchy, keyword in H1, scannable sections                   |
| **Schema opportunities** | No schema planned                       | Basic Organization schema only | Full schema coverage (Organization, WebSite, FAQ, Breadcrumb, Service)     |
| **Technical foundation** | No robots.txt, no sitemap, no canonical | Some present but incomplete    | robots.txt, sitemap.xml, canonical, manifest.json all present and correct  |
| **OG / social sharing**  | No OG tags                              | Basic og:title only            | Full OG + Twitter card on every page with page-specific title, desc, image |
| **Core Web Vitals**      | Poor performance, layout shifts         | Meets thresholds on some pages | LCP < 2.5s, INP < 200ms, CLS < 0.1 on all pages                            |

A page is not ready for launch below an average of 4.
