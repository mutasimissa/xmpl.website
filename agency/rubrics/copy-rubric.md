# Copy Rubric

Score each item from 1 to 5.

| Criterion                 | 1 (poor)                                 | 3 (acceptable)                          | 5 (excellent)                                          |
| ------------------------- | ---------------------------------------- | --------------------------------------- | ------------------------------------------------------ |
| **Clarity**               | Confusing, jargon-heavy, unclear purpose | Understandable but requires effort      | Immediately clear, scannable, no ambiguity             |
| **Relevance to audience** | Generic, could be for anyone             | Addresses the audience but loosely      | Speaks directly to the buyer's role, pain, and context |
| **Differentiation**       | Interchangeable with any competitor      | Some unique elements but weak           | Clearly communicates what sets this business apart     |
| **Trust and proof**       | No proof or trust signals                | Some proof mentioned but not specific   | Specific, credible evidence placed near key claims     |
| **CTA strength**          | No CTA or vague ("Learn more")           | CTA present but generic                 | Outcome-oriented, specific, and well-placed            |
| **Readability**           | Dense paragraphs, no structure           | Some structure, could be more scannable | Short paragraphs, bullet points, clear headings        |
| **Brand tone fit**        | Tone contradicts brand strategy          | Mostly on-brand with some drift         | Consistently matches the defined tone of voice         |

A page is not ready for launch below an average of 4.
